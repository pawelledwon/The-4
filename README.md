# The 4 – 3D Cooperative Platform Game

## Overview
**The 4** is a three-dimensional cooperative platform game designed for up to **four local players**.  
The game focuses on teamwork, puzzle solving, and overcoming environmental obstacles through collaboration between players.

This project was developed as part of an **Engineering Bachelor's Thesis** in Computer Science.

Players control unique characters with different abilities and must work together to complete levels, solve puzzles, and navigate challenging platforming sections.

---

## 🎓 Engineering Thesis

This project was created as a **final engineering project (Bachelor's Thesis)** at the Faculty of Automatic Control, Electronics and Computer Science.

**Thesis title:**  
*Project and Implementation of a Three-Dimensional Platform Game with Cooperation Mode*

The thesis covers:
- analysis of platform games and cooperative mechanics
- system requirements and design
- implementation details
- architecture and key classes
- testing and validation of the game

---

## 🎮 Game Description

The game supports **local cooperative gameplay for up to four players on a single device**.  
Each player controls a different character with unique attributes and abilities.

Players must collaborate to:
- solve environmental puzzles
- move objects
- build structures
- navigate obstacle courses
- reach checkpoints and complete levels

The gameplay emphasizes **communication, coordination, and creative problem solving**.

---

## 🧍 Playable Characters

Each character has different abilities:

### Small
- Highest jump
- Fastest movement speed
- Can lift only small objects

### Default
- Balanced abilities
- Medium jump and movement speed
- Can lift standard objects

### Long
- Tall character with long arms
- Lower jump
- Can lift most objects

### Muscle
- Strongest character
- Slowest movement
- Can lift all objects

## 🗺️ Game Levels

The game contains **five progressively more difficult levels**:

1. **Tutorial Level**
   - Introduces controls and mechanics
   - Basic platforming challenges

2. **Cooperation Level**
   - Requires stacking objects
   - Players must activate buttons together

3. **Puzzle Level**
   - Logical puzzles with objects
   - Moving platforms obstacle course

4. **Advanced Challenge Level**
   - Ice platforms
   - Cannon obstacles
   - Bridge building mechanics

5. **Final Level**
   - Combination of previous mechanics
   - Math puzzles for individual players
   - Final cooperative challenges

## 🎥 Gameplay Mechanics

Players can:
- walk and jump
- push, lift, and throw objects
- interact with environmental elements
- activate switches and mechanisms
- cooperate to solve puzzles

The camera dynamically adjusts to ensure all players remain visible on screen.
